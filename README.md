# Human-X-AI-benchmark

# 📌 Project Goal

This project was created to benchmark three different software development approaches:

## 🤖 Full AI
The application is fully generated by AI with minimal human intervention.

Focus:
- Maximum speed
- AI-generated architecture
- AI-generated decisions

---

## 🧠 Full Human
The application is entirely developed manually without AI assistance.

Focus:
- Human problem solving
- Architecture decisions
- Manual implementation

---

## ⚡ Human + AI
The application is developed collaboratively between human and AI.

Focus:
- Productivity
- Architectural thinking
- AI-assisted acceleration
- Human supervision and decision making

---

# 📊 Benchmark Metrics

The comparison evaluates:

- Development speed
- Code quality
- Readability
- Maintainability
- Scalability
- Debugging difficulty
- Architectural consistency
- Developer experience
